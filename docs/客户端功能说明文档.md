# BDCS2 游戏平台 — 客户端功能说明文档

> 本文档面向运营人员，详细介绍 BDCS2 游戏平台客户端各功能模块的操作流程与业务逻辑。

---

## 一、登录与注册

### 1.1 手机号登录/注册

用户打开平台后进入登录页面，页面采用赛博朋克风格设计，展示游戏角色形象和 BDCS2 品牌 LOGO。登录方式为**手机号 + 短信验证码**，流程如下：

1. 用户输入手机号（默认区号 +86），点击"获取验证码"按钮
2. 系统向该手机号发送 6 位数字验证码（有效期 5 分钟）
3. 用户输入验证码，勾选"已阅读并同意《用户服务协议》和《隐私政策》"
4. 点击"立即登录 / 注册"按钮完成登录

如果该手机号尚未注册，系统将自动创建新账号。

### 1.2 新用户设置昵称与头像

新注册用户首次登录后，系统弹出**设置昵称与头像**弹窗，用户必须完成以下操作才能进入游戏：

- **输入昵称**：支持 1-20 个字符，系统实时检查昵称是否已被占用，重复昵称不可使用
- **选择头像**：提供 16 个系统预设头像供选择，用户点击即可选中

完成设置后点击"确认进入游戏"按钮即可进入首页。

## 三、游戏模块

### 3.1 竞技场（Arena）

竞技场是 PVP 对战模式的核心玩法，玩家与其他玩家（或机器人）进行对战。

**主要功能：**
- 进入竞技场大厅，查看可用的对战房间
- 选择房间加入对战
- 对战结束后查看结果和奖励
- 竞技场历史记录（/arena-history）可查看过往对战记录

### 3.2 幸运转盘 ROLL-X

幸运转盘是一个倍率博弈游戏，玩家设置投注金额和目标倍率，转盘旋转后根据结果决定输赢。

**游戏流程：**
1. 玩家设置投注金额（在最小/最大投注额范围内）
2. 选择目标倍率（倍率越高，中奖概率越低，但奖金越高）
3. 点击旋转按钮，转盘开始旋转
4. 转盘停止后，若指针落在绿色区域则中奖，获得投注金额 x 倍率的奖金
5. 若未中奖，扣除投注金额

**关键参数（由管理后台配置）：**

| 参数 | 说明 |
|------|------|
| RTP（返还率） | 控制整体返还比例，默认 96% |
| 最小投注额 | 每次最低投注金额 |
| 最大投注额 | 每次最高投注金额 |
| 最小倍率 | 可选的最低倍率 |
| 最大倍率 | 可选的最高倍率 |

### 3.3 过马路（Rush）

闯关类小游戏，玩家需要控制角色安全通过障碍物。

### 3.4 ROLL 房

ROLL 房是多人参与的抽奖活动，由管理员创建房间并设置奖品，玩家加入后等待开奖。

**玩家操作流程：**
1. 在 ROLL 房列表页浏览可参与的房间
2. 支持按"全部"、"可参与"、"我参与的"、"已结束"筛选
3. 点击房间查看详情（奖品列表、参与条件、倒计时等）
4. 满足条件后点击"参与"按钮加入
5. 开奖后查看中奖名单

**房间规则：**
- 每个房间有参与门槛（最低充值金额要求）
- 有最大参与人数限制
- 有开始时间和结束时间
- 奖品可以是金币或道具

### 3.5 叮咚（DingDong）

休闲类小游戏。

### 3.6 Vortex

漩涡主题游戏，最新上线的游戏模块。

## 四、背包系统

### 4.1 道具背包（/bag）

背包页面展示玩家通过游戏获得的所有道具，支持以下操作：

| 操作 | 说明 |
|------|------|
| 查看道具 | 展示道具名称、图片、品质等级、价值 |
| 提取道具 | 将道具状态改为"已提取"，等待发货 |
| 回收道具 | 将道具回收为金币，金币直接到账 |

**道具状态流转：**
- **待处理（0）**：刚获得的道具，可提取或回收
- **已提取（1）**：已申请提取，等待发货
- **已回收（2）**：已回收为金币

### 4.2 扩展背包（/backpack）

扩展背包页面，提供更多背包管理功能。

## 五、充值系统

### 5.1 充值页面（/recharge 或 /deposit）

充值页面展示管理后台配置的充值档位列表，每个档位包含：

| 字段 | 说明 |
|------|------|
| 金额 | 需支付的人民币金额 |
| 金币 | 充值后获得的金币数量 |
| 赠送钻石 | 额外赠送的钻石数量 |
| 标签 | 如"热门"、"推荐"等运营标签 |
| 首充标记 | 是否为首充专属档位 |

**充值流程：**
1. 玩家选择充值档位
2. 选择支付方式（支付宝/微信/手动）
3. 提交充值申请，生成订单
4. 管理员在后台审批订单
5. 审批通过后，金币和钻石自动到账

## 八、VIP 系统（/vip）

VIP 系统根据玩家的累计充值金额自动升级 VIP 等级（VIP0-VIP10），不同等级享有不同特权：

| VIP 等级 | 所需充值积分 | 充值返利 | 特权说明 |
|----------|-------------|---------|---------|
| VIP0 | 0 | 0% | 基础权限 |
| VIP1-VIP10 | 逐级递增 | 逐级递增 | 由管理后台配置 |

VIP 等级的升级条件和特权内容均由管理后台动态配置。

## 九、邮件系统（/mailbox）

站内邮件系统用于接收系统通知、活动公告、奖励发放等消息。

**功能：**

## 十、我的记录（/my-records）

记录页面提供三个维度的历史数据查询：

### 10.1 金币流水

展示所有金币变动记录，包括：
- 充值到账、游戏输赢、道具回收、管理员调整等
- 支持按类型筛选和时间范围筛选（今天/昨天/近7天/全部）
- 每条记录显示变动金额、变动后余额、变动原因和时间

### 10.2 提货记录

展示所有已提取的道具记录，包括道具名称、品质、价值、提取时间等。

### 10.3 赠送记录

展示道具赠送相关记录（功能预留，尚未完全实现）。

## 十一、Steam 设置

用于绑定 Steam 账号，方便道具提货时发送到玩家的 Steam 账户。

**设置项：**
- 主号 Steam 交易链接
- 副号 Steam 交易链接
- 绑定码（系统自动生成，用于验证身份）

---

## 十二、安全密码

玩家可设置安全密码，用于保护账户安全。设置流程：
1. 系统向绑定手机号发送验证码
2. 输入验证码验证身份
3. 设置 6-20 位安全密码

**玩家端功能：**

- 主动关闭会话
